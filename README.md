# 🍓 Strawberry Leaf Disease Detection

## 📌 Project Overview
This project uses Deep Learning to detect diseases in strawberry leaves using image classification techniques.  
The model is trained to identify different types of leaf diseases from input images.

---

## 🧠 Technologies Used
- Python
- PyTorch
- CNN (Convolutional Neural Network)
- Jupyter Notebook
- NumPy
- Matplotlib

---

## 📂 Project Structure
- `app.py` – Application script
- `main.py` – Model execution
- `organize_dataset.py` – Dataset preprocessing
- `organize_test.py` – Test data organization
- `organize_val.py` – Validation data organization
- `strawberry.ipynb` – Model training notebook
